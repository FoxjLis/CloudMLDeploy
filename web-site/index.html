Технологии

YOLO 8

В своем проекте мы используем нейронную сеть на базе YOLO 8 с самым тяжелом набором предобученных данных. Этот выбор обеспечивает бесприцидентную точность распознавания объектов на фото. С задачей определения типа имплантов эта сеть справляется на отлично, мы имеем весьма высокие параметры точности, речь о которых пойдет дальше.

Flask

В нашем проекте мы используем Flask. Это позволяет вам обустроить соббственный сервис на нашей базе, используя наш API. WEB приложения на базе Flask работают быстрее, чем аналоги, при этом могут быть достаточно бысто развёрнуты на сервере. Для получения API кода вы можете с нами связаться.
